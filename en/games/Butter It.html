FAQ
I don't know how to install this. ¿Is it safe?

Yup, it is 100% safe. "And why is it not available on Google Play? ¿Why is not available on Steam?" you may ask. Well, because I'm poor and don't have money to pay for the licenses :c


But I will buy the Google Play license, at least; I think so and I hope so. For now, you can trust the APK and EXE as they are avaible on Itch.io which is a very well-known and trusted website.


To install the APK file you may need to allow applications from external sources. Usually, when you try to open the APK without granting the permission, Android will let you activate that setting in a pop-up window. If not, you can activate it manually under 'Security and Privacy' settings.


If it asks to enable 'Play Protect' while installing, you can do whatever you want. I usually keep that disabled.

I found a bug/error/glitch

Oops! Sorry about that. You can use the 'Report Bug' button of the main menu. Or click here.

I don't have a gyroscope :(

The game is designed to be played using gyroscope, but if you don't have one or simply don't like it, there's an option to play using the touch screen. Go to 'Options' in the main menu (or pause the game) and check 'Use touch controls'.

I feel as if my lungs would come out, but the game does not respond!

The game may have problems with Xiaomi devices, which makes it unplayable. You can try some of the following:


  • Make sure you granted microphone permissions. In your device, go to 'Settings > Apps and Notifications (or similar) > Butter It > App permissions (or similar)' and turn the 'Microphone' toggle on.
  • Make sure the microphone sensitivity is not set at 0% in the game options menu. Pause the game, go to 'Options' and set the 'Microphone Sensitivity' slider to something above 0%. 50% is the default.
  • Turn the 'Microphone Sensitivity' slider all the way up.
  • Make sure no other app is blocking or overwriting the microphone use.
  • Try using an external microphone.
  • Use the 'Spacebar' from a keyboard instead of the microphone. Remember to press 'Spacebar' a bunch of times as fast as you can until the red warning disappears.


If you still have problems, fill a bug report here.
