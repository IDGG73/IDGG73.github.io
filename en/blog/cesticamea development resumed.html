Cesticamea Development Resumed

Date: 06/18/2021

OH YES, THAT'S IT BABY. CESTICAMEA HAS RETURNED.


Some days ago I spent some time deleting some files from my PC to free up some space, and I found some files of Cesticamea.

I completely forgot about Cesticamea, really, I don't even know how.


First thing I did was to delete almost every script, why? Because they weren't prepared for multiplayer and they were... kinda... confusing... Keep in mind those scripts where almost 2 years old.


Then I redesigned the logo.



It definitely looks better. It had too much blur, and now it looks more like the rest of the UI.


Now, let's talk about the game itself.

The UI is the same, the backgrounds are the same, the music is the same; the only thing I'm changing is the code.


I was thinking about how a trivia game is funnier with friends, because you have someone to mock at when you knew the answer while they didn't. And the game was being made with only one player in mind. So Cesticamea, being a trivia game, NEEDS multiplayer. At least local multiplayer, but it needs multiplayer.


Of course I want to give it online multiplayer, but it's not that easy. I'll make some tests with Mirror and Photon to see how they perform (I'm pretty sure I'll end up using Mirror). Meanwhile, online multiplayer is not confirmed.


I don't have a release date yet, but be sure that I'm trying to release it as soon as possible at the same time I'm trying to fix all the bugs so you can enjoy the game without having to update it. I'm not saying I won't update it, I'm just saying I won't update it too often.


By the way, the new Unity Input System package is fucking awesome. It's also harder to use via code, but damn, you can change the color of the PS4 controller light, that's fucking cool.
